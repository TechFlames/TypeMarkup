# **TypeMarkup**

TypeMarkup is a beautiful and simple language for web documents.

## **Preview**

```TypeMarkup
@attr lang en
*base html

&base head
  @attr charset UTF-8
  meta

  title TypeMarkup

  @attr http-equiv X-UA-Compatible
  @attr content IE-edge
  meta

  @attr content width=device-width, initial-scale=1.0
  @attr name viewport
  meta

&base body
  h1 TypeMarkup

  p Hello from :TypeMarkup
```
